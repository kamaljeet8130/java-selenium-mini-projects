## Feature: Login Functionality

### Description
This feature allows a registered user to log in using a username and password
to access the secure area of the application.

### User Stories
1. As a user, I want to log in with valid credentials so that I can access the secure area.
2. As a user, I should see an error message if I enter invalid credentials.
3. As a user, I should not be able to log in if required fields are empty.
